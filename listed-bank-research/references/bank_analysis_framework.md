# 银行业基本面分析框架

> 本参考文档基于国信证券、东方证券等券商银行业研究框架整理，供上市银行基本面研究使用。

## 一、银行商业模式核心逻辑

**银行价值 = 承担的风险 + 管理风险的能力**

银行是经营风险的机构，核心商业模式是**通过承担风险来获取收益（利息+手续费）**。

```
资金来源（负债端）          资金运用（资产端）
     │                          │
     ▼                          ▼
吸收存款 ──────────────► 发放贷款和垫款
同业负债                  金融投资（债券/非标）
应付债券                  同业资产
向央行借款                现金及存放央行
     │                          │
     ▼                          ▼
  计息负债                生息资产
```

**盈利来源 = 息差收入 + 中间业务收入 - 信用成本 - 运营成本**

## 二、财务报表分析

### 2.1 资产负债表（Balance Sheet）

#### 资产端（资金运用）

| 科目 | 重要性 | 分析要点 |
|------|--------|----------|
| 现金及存放央行 | ★★★ | 法定存款准备金率（大型银行13%，中小银行10%） |
| 同业资产 | ★★ | 拆放同业、买入返售——利率敏感，流动性管理工具 |
| 发放贷款和垫款 | ★★★★★ | **核心生息资产**，分析结构（对公/零售/票据）、行业分布、期限、担保方式 |
| 金融投资 | ★★★★ | 债券（利率债/信用债）、非标、基金——分析配置策略和信用风险 |
| 以公允价值计量且变动计入当期损益的金融资产(FVPL) | ★★ | 交易盘，波动计入利润表 |

#### 负债端（资金来源）

| 科目 | 重要性 | 分析要点 |
|------|--------|----------|
| 吸收存款 | ★★★★★ | **最稳定资金来源**，分析成本率（活期/定期、对公/零售） |
| 同业负债 | ★★★ | 稳定性差、利率敏感，占比过高=流动性风险信号 |
| 应付债券 | ★★★ | 金融债、资本债——中长期资金来源 |
| 向央行借款 | ★★ | OMO/MLF等货币政策工具 |

#### 所有者权益

| 科目 | 重要性 | 分析要点 |
|------|--------|----------|
| 实收资本/股本 | ★★★★★ | 融资能力基础 |
| 资本公积 | ★★★ | 溢价发行 |
| 盈余公积 | ★★★ | 利润积累 |
| 未分配利润 | ★★★★★ | 核心内源资本来源 |
| 贷款损失准备 | ★★★★★ | 拨备余额，资产质量的缓冲垫 |

### 2.2 利润表（Income Statement）

```
营业收入
├── 利息净收入 (Net Interest Income)
│   ├── 利息收入：贷款利息收入 + 金融投资利息收入 + 同业利息收入
│   └── 利息支出：存款利息支出 + 同业负债利息支出 + 债券利息支出
├── 手续费及佣金净收入 (Net Fee & Commission Income)
│   ├── 银行卡手续费
│   ├── 结算与清算手续费
│   ├── 托管业务收入
│   ├── 理财业务收入
│   └── 投行业务收入
└── 其他非息收入
    ├── 公允价值变动收益（FVPL）
    ├── 投资收益
    └── 汇兑收益

营业支出
├── 业务及管理费 (Operating Expenses)
│   ├── 员工费用
│   ├── 业务费用
│   └── 折旧与摊销
├── 信用减值损失 (Credit Impairment Losses)
│   └── 贷款减值损失 + 金融投资减值损失
└── 其他支出

净利润 = 营业收入 - 营业支出 - 所得税
```

### 2.3 关键财务指标计算

#### 盈利能力指标

| 指标 | 公式 | 参考标准 |
|------|------|----------|
| **净息差 (NIM)** | 利息净收入 / 平均生息资产 × 100% | 国有大行 ~1.8%，股份制 ~2.0-2.5%，优质区域行 ~2.5%+ |
| **ROE** | 净利润 / 平均净资产 × 100% | >12% 为优质，>15% 为优秀 |
| **ROA** | 净利润 / 平均总资产 × 100% | >0.8% 为优质 |
| **成本收入比** | 业务及管理费 / 营业收入 × 100% | <35% 为优秀，35-50% 为正常，>50% 需关注 |
| **非息收入占比** | 非利息收入 / 营业收入 × 100% | >30% 为多元化收入结构 |

#### 杜邦分析（DuPont Decomposition）

```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利润/营业收入) × (营业收入/平均总资产) × (平均总资产/平均净资产)

     ┌──────────────┐
     │   ROE       │  = 盈利能力 × 资产效率 × 杠杆水平
     └──────┬───────┘
            │
     ┌──────┴───────┐
     ▼              ▼
  净利润率       资产周转率
(净息差+非息)    (生息资产利用率)
                  │
            权益乘数
           (负债水平)
```

#### 资产质量指标

| 指标 | 公式 | 参考标准 |
|------|------|----------|
| **不良贷款率 (NPL Ratio)** | 不良贷款余额 / 贷款总额 × 100% | <1.5% 为优质，1.5-2% 为正常，>2% 需警惕 |
| **拨备覆盖率** | 贷款损失准备 / 不良贷款 × 100% | 监管要求 ≥150%，>200% 为充足 |
| **贷款拨备率** | 贷款损失准备 / 贷款总额 × 100% | >2.5% 为充足 |
| **关注类贷款占比** | 关注类贷款 / 贷款总额 × 100% | 趋势比绝对值更重要 |
| **重组贷款占比** | 重组贷款 / 贷款总额 × 100% | 过高可能隐藏不良 |

#### 贷款五级分类

| 分类 | 定义 | 拨备要求 |
|------|------|----------|
| 正常类 | 借款人能履行合同，无迹象表明不能按时足额偿还 | 1% |
| 关注类 | 尽管有能力偿还，但存在可能影响偿还的不利因素 | 2-3% |
| 次级类 | 借款人的还款能力出现明显问题，依靠正常经营收入无法保证足额偿还 | 25-50% |
| 可疑类 | 借款人无法足额偿还，即使执行担保也会造成较大损失 | 50-75% |
| 损失类 | 在采取所有可能措施后仍无法收回，或只能收回极少部分 | 100% |

#### 资本充足率指标

| 指标 | 公式 | 监管要求（2024） |
|------|------|-----------------|
| **核心一级资本充足率 (CET1)** | 核心一级资本 / 风险加权资产 × 100% | ≥ 7.5% + 储备资本 2.5% = 10%（系统性重要银行更高） |
| **一级资本充足率** | (核心一级+其他一级资本) / RWA × 100% | ≥ 8.5% |
| **资本充足率 (CAR)** | 总资本 / 风险加权资产 × 100% | ≥ 10.5% |
| **杠杆率** | 一级资本 / 调整后表内外资产余额 × 100% | ≥ 4% |

#### 流动性指标

| 指标 | 公式 | 监管要求 |
|------|------|----------|
| **流动性比例** | 流动性资产 / 流动性负债 × 100% | ≥ 25% |
| **流动性覆盖率 (LCR)** | 高质量流动性资产 / 30天净现金流出 × 100% | ≥ 100% |
| **净稳定资金比例 (NSFR)** | 可用稳定资金 / 所需稳定资金 × 100% | ≥ 100% |
| **存贷比 (LDR)** | 贷款总额 / 存款总额 × 100% | 已取消硬性限制，监测指标（通常 < 80% 为稳健） |

## 三、利息收入分析框架

### 3.1 生息资产结构分析

```
生息资产 = 贷款净额 + 金融投资 + 同业资产 + 存放央行 + ...

分析维度：
├── 按产品类型：
│   ├── 公司贷款（一般贷款、票据贴现）
│   ├── 个人贷款（住房贷款、消费贷款、经营贷款、信用卡）
│   └── 贸易融资
├── 按担保方式：
│   ├── 信用贷款
│   ├── 担保贷款
│   └── 抵押/质押贷款
├── 按行业分布：
│   ├── 制造业
│   ├── 房地产（高集中度=风险）
│   ├── 地方政府融资平台
│   ├── 批发零售业
│   └── 个人住房贷款
└── 按期限结构：
    ├── 短期贷款（<1年）
    └── 中长期贷款（≥1年）
```

### 3.2 计息负债成本分析

```
计息负债 = 存款 + 同业负债 + 应付债券 + 向央行借款 + ...

存款成本率 = 存款利息支出 / 平均存款余额 × 100%

分析维度：
├── 按期限：
│   ├── 活期存款（成本最低，0.3-0.5%）
│   ├── 定期存款（1-5年期，成本1.5-3%）
│   └── 通知存款
├── 按客户：
│   ├── 公司存款
│   └── 个人存款
└── 趋势：
    ├── 活期占比提升 → 负债成本改善
    └── 定期化严重 → 负债成本压力
```

### 3.3 净息差 (NIM) 驱动因素

```
NIM = (利息收入 - 利息支出) / 平均生息资产
    = (生息资产收益率) - (计息负债成本率) × (负债/资产比率)

驱动因素分解：
┌─────────────────────────────────────────────────────┐
│                    NIM                              │
├─────────────────────────────────────────────────────┤
│ 资产端收益           │ 负债端成本       │ 结构因子   │
├──────────────────────┼─────────────────┼────────────┤
│ 贷款收益率↑          │ 存款成本率↓      │ 负债/资产比 │
│ (定价能力↑)          │ (主动负债↓)      │            │
│                      │                 │            │
│ 资产结构↑            │ 同业负债成本↓    │            │
│ (高收益资产占比↑)    │                 │            │
└──────────────────────┴─────────────────┴────────────┘
```

## 四、非息收入分析框架

### 4.1 手续费及佣金收入结构

| 业务类型 | 收入特征 | 趋势判断 |
|----------|----------|----------|
| **银行卡手续费** | 信用卡收单、分期手续费 | 与消费信贷正相关 |
| **结算与清算** | 支付结算、现金管理 | 稳定，与企业活跃度相关 |
| **托管业务** | 基金托管、资产托管 | 依赖AUM规模和市场情绪 |
| **理财业务** | 管理费+超额业绩提成 | 资管新规后转型，净值化 |
| **投行业务** | 债券承销、并购咨询 | 与资本市场活跃度相关 |
| **代理业务** | 保险、基金代销 | 依赖渠道和客户粘性 |

### 4.2 非息收入质量评估

```
非息收入质量评估维度：

1. 占比：非息收入/营业收入
   - < 20%：利息收入主导
   - 20-30%：转型期
   - > 30%：多元化银行

2. 稳定性：
   - 银行卡、托管、结算 → 相对稳定
   - 理财、投行 → 波动较大（市场相关）

3. 增速：与利息收入增速对比
   - 非息增速持续高于利息 → 业务转型成功
   - 非息增速大幅波动 → 依赖市场贝塔

4. 科技含量：判断是否为真正轻资本转型
   - 低科技含量：通道业务（监管打压对象）
   - 高科技含量：金融科技输出、数字化服务
```

## 五、资产质量分析框架

### 5.1 不良贷款生成路径

```
正常贷款 → 关注类 → 次级类 → 可疑类 → 损失类
   │                      ↑
   └──► 重组贷款 ─────────┘

不良生成率 = 本期新增不良 / 期初正常类贷款 × 100%
            (衡量新生成的不良)

不良净生成率 = (期末不良 - 期初不良 + 本期核销) / 期初贷款余额 × 100%
             (更准确的真实不良生成指标)
```

### 5.2 拨备充足性评估

```
评估方法：

1. 静态指标：拨备覆盖率
   - 150-200%：基本充足
   - 200-300%：充足
   - > 300%：非常充足（可能隐藏利润）

2. 动态趋势：拨备变化趋势
   - 拨备覆盖率持续下降 → 资产质量恶化或隐藏利润
   - 拨备覆盖率持续上升 → 审慎经营或业绩释放

3. 跨周期对比：经济下行期的拨备策略
   - 逆周期银行：上行期多提拨备，下行期少提（平滑利润）
   - 顺周期银行：随周期波动，不平滑

4. 重组贷款分析：
   - 重组贷款认定标准是否严格
   - 重组后是否真正改善还款能力
```

### 5.3 行业集中度风险

| 行业 | 风险特征 | 关注要点 |
|------|----------|----------|
| **房地产** | 高杠杆、周期性强、抵押品价值波动大 | 按揭+对公+理财投资全口径敞口 |
| **地方政府融资平台** | 政策敏感、透明度低 | 隐债识别、财政能力绑定 |
| **制造业** | 产能周期、出口依赖 | 贸易摩擦、产业链转移 |
| **批发零售业** | 抗风险能力弱 | 疫情、消费降级 |
| **建筑业** | 依赖地产链 | 与房地产高度关联 |

## 六、资本管理分析框架

### 6.1 资本充足率达标路径

```
内源资本：利润留存 → 核心一级资本 ↑

外源资本：
├── 一级资本工具：永续债（其他一级资本）
└── 二级资本工具：二级资本债

股权融资：配股、增发（最直接但摊薄EPS）

RWA管理：
├── 优化资产结构（低风险权重资产占比↑）
└── 压缩高风险权重资产
```

### 6.2 资本规划能力评估

```
评估问题：

1. 增长潜力：
   - 当前资本能否支撑未来2-3年资产增长？
   - RWA增速 vs 资本净额增速

2. 分红政策：
   - 分红率是否合理（30%为常见中枢）
   - 高分红是否侵蚀核心资本

3. 融资计划：
   - 是否公告融资计划
   - 融资方式和时机判断

4. 系统重要性：
   - 全球系统重要性银行（G-SIBs）附加资本1-3.5%
   - 国内系统重要性银行（D-SIBs）附加资本0.25-1.5%
```

## 七、流动性风险管理框架

### 7.1 流动性风险识别

```
预警信号：

1. 结构性指标恶化：
   - 存贷比持续上升
   - 同业负债占比 > 30%
   - 长期资产占比过高

2. 边际融资成本上升：
   - 同业存单发行利率上行
   - 信用利差扩大
   - 发行成功率下降

3. 市场信号：
   - CDS利差扩大
   - 评级下调
   - 负面舆情

4. 监管指标：
   - LCR持续接近100%
   - NSFR持续下降
```

### 7.2 流动性应急计划

```
银行应具备的流动性应急能力：

1. 高质量流动性资产 (HQLA) 缓冲
   - 国债、政策性金融债（折算率95-100%）
   - 央行存款（折算率100%）
   - AA级以上信用债（折算率50%）

2. 应急融资渠道
   - 央行借贷便利（SLF/MLF）
   - 同业拆借额度
   - 债券质押式回购

3. 资产负债期限结构
   - 避免过度期限错配
   - 保持合理的中长期贷款占比
```

## 八、风险因素识别清单

### 8.1 宏观风险

| 风险类型 | 影响路径 | 观察指标 |
|----------|----------|----------|
| GDP增速下行 | 贷款需求↓ + 违约率↑ | PMI、CPI、PPI |

| 汇率波动 | 结售汇+外币资产 | 美元/人民币 |

| 风险类型 | 影响路径 | 观察指标 |
|----------|----------|----------|
| 房地产风险 | 按揭+对公敞口 | 房价指数、地产销售 |
| 地方债务风险 | 城投贷款+地方债 | 隐债规模、财政数据 |
| 制造业风险 | 对公贷款 | 制造业PMI、出口数据 |
| 消费风险 | 信用卡+消费贷 | 居民杠杆率、消费增速 |

### 8.3 公司特有风险

| 风险类型 | 观察信号 |
|----------|----------|
| 贷款激进 | 贷款增速远高于行业 |
| 资产质量恶化 | 不良率↑ + 关注率↑ |
| 负债成本上升 | 存款流失 + 同业依赖↑ |
| 内控失效 | 大额罚单 + 案件频发 |

| 战略失误 | 盲目扩张+新业务亏损 |

## 九、同业比较框架

### 9.1 估值比较指标

| 指标 | 计算 | 用途 |
|------|------|------|
| P/B (市净率) | 股价 / 每股净资产 | 银行估值最常用指标 |
| P/E (市盈率) | 股价 / 每股收益 | 盈利效率比较 |
| EV/EBITDA | 企业价值/EBITDA | 剔除资本结构影响 |

### 9.2 核心指标同业对标

```
比较维度：

1. 盈利能力：
   - ROE排序（高ROE = 股东回报好）
```
